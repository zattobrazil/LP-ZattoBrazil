# 📁 Pasta de Imagens

Coloque aqui as imagens do projeto:

## ✅ Imagens Necessárias:

### **Hero Section**
- `img-header.jpg` - Imagem de fundo da seção Hero (recomendado: 1920x1080px)

### **Landing Pages**
- `agro-hero.jpg` - Imagem para o nicho Agro
- `tech-hero.jpg` - Imagem para o nicho Tecnologia
- `eventos-hero.jpg` - Imagem para o nicho Eventos
- `agro-about.jpg` - Imagem "Sobre" para Agro
- `tech-about.jpg` - Imagem "Sobre" para Tecnologia
- `eventos-about.jpg` - Imagem "Sobre" para Eventos

### **Open Graph (Redes Sociais)**
- `og-image.jpg` - Imagem compartilhamento (1200x630px)

## 📐 Especificações Recomendadas:

- **Hero:** 1920x1080px (Full HD landscape)
- **About:** 1200x800px (4:3 ratio)
- **OG Image:** 1200x630px (exato para Facebook/LinkedIn)
- **Formato:** JPG ou PNG
- **Tamanho:** < 500KB (otimizado para web)

## 🔧 Como Usar:

1. Coloque a imagem nesta pasta
2. Referencie no código como: `/images/nome-da-imagem.jpg`
3. O Next.js serve automaticamente a partir da pasta `public`

## 🎨 Dica:

Enquanto não tem as imagens reais, você pode usar:
- [Unsplash](https://unsplash.com) - Fotos gratuitas de alta qualidade
- [Pexels](https://pexels.com) - Banco de imagens livre
